# PARC Fitness Timer

An Android fitness timer application built with Kotlin and Jetpack Compose.

## Features
- Real-time fitness tracking timer
- Modern Android UI using Jetpack Compose
- Modern Architecture out of the box

## Requirements
- Minimum SDK: 26
- Target SDK: 34
- Android Studio Ladybug or newer.

## Building and Running
1. Open the project in Android Studio.
2. Wait for Gradle sync to complete.
3. Build and Run the `app` configuration on your emulator or physical device.
